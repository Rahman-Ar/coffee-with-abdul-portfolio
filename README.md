## 🚀 Getting Started

### Clone the repo
```bash
git clone https://github.com/Rahman-Ar/coffee-with-abdul-portfolio.git
cd coffee-with-abdul-portfolio

Install dependencies
npm install

Run locally
npm start

Build for production
npm run build

```
